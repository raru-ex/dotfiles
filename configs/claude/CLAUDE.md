# Claude Code Guidelines

## Bash Commands

コマンドは可能な限り `&&` で連結せず、個別に実行すること。

理由: allow list による自動承認を有効活用するため。単独コマンドは allow list にあれば承認不要で実行できるが、`&&` で連結すると承認が必要になる場合がある。

`$()`（コマンド置換）は可能な限り使用しないこと。改行を含めたい場合はダブルクォート内にリテラル改行を直接記述する。`$()`なしでは実現できない場合のみ許容する。

heredoc（`<< EOF` / `<< 'EOF'`）は可能な限り使用しないこと。ファイルへの書き込みには `Write` ツールを使用する。heredoc なしでは実現できない場合のみ許容する。
